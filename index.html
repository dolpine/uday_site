About

UDAY is a single creative.

I create a beautiful, original, amazing things.

I am working as a software engineer in New Delhi, India. I am broadly interested in web application development. While the Internet has reached a large number of users, a growing pain point that hinders it’s reach is the lack of accessibility mechanisms. I believe Internet should be open to all – no individual should face any hindrance while accessing the Internet under any circumstances. That's why, I like to work on web development. Another related problem here is that of interoperability across devices. Modern web enabled devices come in all shapes and sizes, this makes it difficult for web developers to gain the most out of a larger screen size while maintaining a sane code base. I believe, once an web app is designed and developed, it should run on all devices while taking full advantage of the variations in screen sizes – that's why I am interested in Responsive Design. Apart from web development, my interest lies in algorithms. I take a keen interest in understanding and implementing various algorithms.
